## Installation
*[You'll need Node.js](https://nodejs.org/)*

1. Create a file named `.env` and write this :

```
PORT=YOUR_PORT
DB_USER=MONGO_USERNAME
DB_PASS=MONGO_PASSWORD
```

2. In a command prompt, use `npm i`


3. To start the dev mode, use `npm start`
